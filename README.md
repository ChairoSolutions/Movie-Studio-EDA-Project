# Movie Dataset Analysis Project

## Project Overview
This project explores a dataset containing information about movies, including titles, genres, ratings, and revenue. The goal was to perform data cleaning, exploratory data analysis (EDA), and prepare insights for visualization and reporting.

1. **Data Inspection and Cleaning**
   - Checked data types and handled missing values.
   - Identified and removed duplicate rows.
   - Ensured columns of interest were correctly formatted for analysis.

2. **Exploratory Data Analysis (EDA)**
   - Summarized numerical and categorical features.
   - Prepared a cleaned version of the dataset suitable for Tableau or other visualization tools.

3. **Notebook Handling**
   - Ensured that all Jupyter notebooks were properly structured and functional.
   - Maintained version control to avoid corrupted files during collaboration.

4. **Documentation**
   - Added clear comments in the notebook for clarity.
